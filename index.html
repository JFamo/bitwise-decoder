Bitwise XOR Decryptor

by J.Famous

This program is the intellectual property of J.Famous 2017.
It will decode a string encoded using the Bitwise XOR operator.
Some variables in this program are assumed, but can be replaced as necessary.

You can input either text directly from the code of a file,
Or manually input elements, seperated by spaces.
Be sure that they are seperated by spaces, and not seperated by new lines!
This program is capable of decoding any number or string, which is any character represented in Unicode.
Most common encoded items will be a hexidecimal value from 40 to 5f.
